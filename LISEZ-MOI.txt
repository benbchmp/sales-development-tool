========================================
  LeadFinder – Guide de démarrage
========================================

LeadFinder te permet de trouver des entreprises sur Google Maps
qui n'ont pas encore de site web (prospects potentiels).


AVANT DE COMMENCER
------------------

1. Tu as besoin d'une clé Google API (Benjamin te la donnera).

2. Dans le dossier du projet, tu trouveras un fichier ".env.example".
   - Fais une copie de ce fichier
   - Renomme la copie en ".env" (sans le .example)
   - Ouvre le fichier .env avec TextEdit
   - Remplace COLLE_TA_CLE_GOOGLE_ICI par la clé que Benjamin t'a donnée
   - Sauvegarde et ferme le fichier


LANCER L'APPLICATION
--------------------

1. Ouvre le Terminal (voir le fichier INSTRUCTIONS_MAC.txt si besoin)

2. Tape cette commande pour aller dans le dossier du projet :
   cd ~/Desktop/LeadFinder

   (Adapte le chemin si tu as mis le dossier ailleurs)

3. Tape cette commande pour lancer :
   bash run.sh

4. Le navigateur s'ouvrira tout seul au bout de quelques secondes.

5. Si c'est la première fois, l'installation prend 1 à 2 minutes.


UTILISER L'APPLICATION
----------------------

1. Tape une ville (ex: Lyon, Paris, Marseille)
2. Choisis un type d'activité dans la liste
3. Clique sur "Rechercher"
4. Les résultats s'affichent dans le tableau
5. Par défaut, seules les entreprises SANS site web sont affichées
6. Clique sur "Export CSV" pour sauvegarder les résultats


ARRÊTER L'APPLICATION
---------------------

Dans le Terminal, appuie sur Ctrl+C (les deux touches en même temps).


EN CAS DE PROBLÈME
-------------------

- "Python n'est pas installé" → Va sur python.org et installe Python 3
- "Le fichier .env est manquant" → Relis la section "Avant de commencer"
- "Erreur API" → Vérifie que la clé Google dans le .env est correcte
- L'appli ne se lance pas → Vérifie que tu es bien dans le bon dossier
